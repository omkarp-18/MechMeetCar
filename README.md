On-Demand Mechanic Service Platform

- Overview -

MechMeetCar is a location-based freelancing platform that connects vehicle owners with nearby verified mechanics for repairs, servicing, and emergency breakdown assistance.
The system brings structure, transparency, and speed to vehicle servicing by enabling real-time booking, job tracking, and trust-based ratings.

- Key Features -

Location-based mechanic discovery
Customer & mechanic authentication
Service request and acceptance flow
Real-time job status tracking
Ratings and feedback system

- Tech Stack -

Frontend: HTML, CSS, JavaScript (Responsive)
Backend: Firebase / REST-based architecture
Database: Firebase / MySQL
Auth: Email-based authentication
